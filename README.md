## About

A modern, fully-featured portfolio website for **Mashoto Bayne Rababalela** — a multifaceted professional bridging technical engineering and strategic business growth.

### Sections
- **Hero** — Personalized intro with floating skill badges
- **About** — Professional summary with parallax photo
- **Skills** — 4 discipline categories (Web Dev, Electrical Eng., Growth & Sales, Leadership)
- **Certificates** — 4 Google certifications with lightbox viewer
- **Contact** — Formspree contact form + LinkedIn CTA + newsletter

### Features
- ⚡ PWA-ready with `manifest.json`
- 🎨 Dark/Light/System theme toggle
- 🔐 Security headers via `vercel.json`
- 📱 Fully responsive — mobile-first design
- 🎞️ Framer Motion scroll animations
- 🔍 Full Open Graph + Twitter Card meta tags
- ♿ Semantic HTML + ARIA labels
- 🚀 Deployed via Vercel
